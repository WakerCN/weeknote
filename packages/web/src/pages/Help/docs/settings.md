# 设置配置 ⚙️

WeekNote 提供了丰富的设置选项，让你可以根据个人需求定制使用体验。

---

## 1. 进入设置页面

### 方式一：导航栏

点击顶部导航栏的「设置」图标。

### 方式二：用户菜单

点击右上角用户头像，选择「设置」。

---

## 2. 模型与 API Key

### 2.1 配置 API Key

**为什么需要 API Key？**

WeekNote 使用 AI 模型生成周报，需要配置至少一个平台的 API Key。

**支持的平台**：

| 平台 | 获取方式 | 说明 |
|------|----------|------|
| **硅基流动** | [siliconflow.cn](https://siliconflow.cn) | 🆓 免费，推荐新手 |
| **DeepSeek** | [deepseek.com](https://deepseek.com) | 💰 付费，性价比高 |
| **OpenAI** | [platform.openai.com](https://platform.openai.com) | 💰 付费，GPT-4o |
| **豆包** | [console.volcengine.com](https://console.volcengine.com) | 💰 付费，需配置接入点 |

**配置步骤**：

1. 进入「模型与 API Key」页面
2. 选择要配置的平台
3. 输入 API Key
4. 点击「保存」

![API Key 配置页面](/help/images/settings/01-apikey-settings.png)

> 🔒 **安全提示**：API Key 会加密存储，只有你可以看到。

### 2.2 选择默认模型

配置 API Key 后，选择一个默认模型：

- **免费用户**：推荐「通义千问 2.5 (7B)」
- **付费用户**：可根据需求选择

### 2.3 模型说明

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| 通义千问 2.5 (7B) | 免费、中文优化 | 日常使用 |
| GLM-4 (9B) | 免费、中文优化 | 中文内容 |
| DeepSeek Chat | 性价比高 | 高质量生成 |
| GPT-4o | 能力强 | 复杂场景 |
| GPT-4o-mini | 轻量版 | 快速生成 |

## 3. Prompt 模板

### 3.1 什么是 Prompt 模板？

Prompt 模板定义了周报的格式和风格，影响 AI 生成的内容。

### 3.2 管理模板

**查看模板**：

1. 进入「Prompt 模板」页面
2. 查看系统模板和自定义模板
3. 点击模板查看详情

![Prompt 模板管理](/help/images/settings/02-prompt-settings.png)

**创建模板**：

1. 点击「新建模板」按钮
2. 填写模板名称和内容
3. 保存模板

**编辑模板**：

1. 点击模板右侧的「编辑」按钮
2. 修改模板内容
3. 保存更改

**删除模板**：

1. 点击模板右侧的「删除」按钮
2. 确认删除

> ⚠️ **注意**：系统模板无法删除，只能编辑。

### 3.3 模板变量

你可以在模板中使用以下变量：

- `{dailyLog}`：Daily Log 内容
- `{dateRange}`：日期范围
- `{userName}`：用户名

**示例**：

```markdown
请将以下 Daily Log 整理为周报：

{dailyLog}

日期范围：{dateRange}
用户：{userName}
```

## 4. 提醒设置

### 4.1 配置推送渠道

WeekNote 支持多种推送渠道：

| 渠道 | 配置项 | 获取方式 |
|------|--------|----------|
| **钉钉机器人** | Webhook + 加签密钥 | 群设置 → 智能群助手 |
| **Server酱** | SendKey | [sct.ftqq.com](https://sct.ftqq.com) |

**配置步骤**：

1. 进入「提醒设置」页面
2. 选择推送渠道
3. 填写配置信息
4. 点击「测试推送」验证配置
5. 保存设置

![提醒设置页面](/help/images/settings/03-reminder-settings.png)

### 4.2 设置提醒时间

你可以设置多个提醒时间点：

- **上班提醒**：如 9:00
- **下班提醒**：如 18:00
- **自定义时间**：任意时间点

### 4.3 提醒功能特性

- ✅ **工作日提醒**：仅在工作日推送
- ✅ **节假日识别**：自动识别法定节假日
- ✅ **调休支持**：支持调休工作日
- ✅ **测试推送**：可以测试配置是否正确

1. 进入「账号管理」页面

## 常见问题

**Q: API Key 安全吗？**

A: 是的。API Key 会加密存储，只有你可以看到。

**Q: 可以删除系统模板吗？**

A: 不可以。系统模板无法删除，但可以编辑。

**Q: 提醒功能不工作怎么办？**

A: 请检查：
1. 配置信息是否正确
2. 网络连接是否正常
3. 是否在工作日

**Q: 可以导出设置吗？**

A: 目前不支持导出设置，但数据会保存在云端。
