# 快速开始 🚀

欢迎使用 WeekNote！本指南将帮助你快速上手，完成账号注册、登录和首次配置。

---

## 1. 注册账号

### 步骤一：访问注册页面

1. 打开 WeekNote 应用
2. 点击「注册」按钮或直接访问注册页面

![登录/注册页面](/help/images/quick-start/01-auth-page.png)

### 步骤二：填写注册信息

你需要提供以下信息：

- **邮箱地址**：用于登录和接收验证码
- **用户名**：显示名称（2-20 个字符）
- **密码**：至少 8 位，包含字母和数字

> 💡 **提示**：WeekNote 支持验证码登录，首次登录时会自动创建账号，无需单独注册。

### 步骤三：完成注册

点击「注册」按钮，系统会自动发送验证邮件到你的邮箱。

---

## 2. 登录系统

WeekNote 支持两种登录方式：

### 方式一：验证码登录（推荐）

1. 输入你的邮箱地址
2. 点击「发送验证码」
3. 查收邮件中的 6 位验证码
4. 输入验证码完成登录

### 方式二：密码登录

1. 输入你的邮箱地址和密码
2. 可选择「记住我」保持登录状态
3. 点击「登录」

---

## 3. 首次配置

登录成功后，你需要完成以下配置才能开始使用：

### 3.1 配置 API Key

**为什么需要 API Key？**

WeekNote 使用 AI 模型来生成周报，你需要配置至少一个平台的 API Key。

**支持的平台**：

| 平台 | 类型 | 说明 |
|------|------|------|
| 硅基流动 | 🆓 免费 | 推荐新手使用 |
| DeepSeek | 💰 付费 | 性价比高 |
| OpenAI | 💰 付费 | GPT-4o / GPT-4o-mini |
| 豆包 | 💰 付费 | 火山方舟 |

**配置步骤**：

1. 进入「设置」→「模型与 API Key」
2. 选择你要配置的平台
3. 输入 API Key（可在各平台官网获取）
4. 点击「保存」

![API Key 配置页面](/help/images/settings/01-apikey-settings.png)

> 💡 **提示**：API Key 会加密存储，只有你可以看到。

### 3.2 选择默认模型

配置 API Key 后，选择一个默认模型：

- **免费用户**：推荐使用「通义千问 2.5 (7B)」
- **付费用户**：可根据需求选择 DeepSeek、GPT-4o 等

---

配置完成后，你就可以开始使用 WeekNote 了：

- 📅 **每日记录**：记录你的日常工作内容

## 常见问题

**Q: 忘记密码怎么办？**

A: 在登录页面点击「忘记密码」，系统会发送重置密码的邮件。

**Q: 收不到验证码邮件？**

A: 请检查垃圾邮件文件夹，或等待 60 秒后重新发送。

**Q: 可以不配置 API Key 吗？**

A: 不可以。WeekNote 需要 AI 模型来生成周报，必须配置至少一个 API Key。
