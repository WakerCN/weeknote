# 常见问题 ❓

这里收集了 WeekNote 使用过程中的常见问题及解答。

---

## 账号相关

### Q1: 忘记密码怎么办？

**A**: 在登录页面点击「忘记密码」，系统会发送重置密码的邮件到你的邮箱。点击邮件中的链接即可重置密码。

### Q2: 收不到验证码邮件？

**A**: 请检查：
1. 垃圾邮件文件夹
2. 邮箱地址是否正确
3. 等待 60 秒后重新发送

如果仍然收不到，请联系客服。

### Q3: 可以修改用户名吗？

**A**: 可以。在「设置」→「账号管理」中可以修改用户名。

## API Key 相关

### Q4: 如何获取 API Key？

**A**: 各平台获取方式：

- **硅基流动**：访问 [siliconflow.cn](https://siliconflow.cn)，注册后获取免费 API Key
- **DeepSeek**：访问 [deepseek.com](https://deepseek.com)，注册后获取 API Key
- **OpenAI**：访问 [platform.openai.com](https://platform.openai.com)，注册后获取 API Key
- **豆包**：访问 [console.volcengine.com](https://console.volcengine.com)，创建应用后获取 API Key

### Q5: API Key 安全吗？

**A**: 是的。API Key 会加密存储，只有你可以看到。我们不会泄露你的 API Key。

### Q6: 可以不配置 API Key 吗？

**A**: 不可以。WeekNote 需要 AI 模型来生成周报，必须配置至少一个 API Key。

### Q7: 推荐使用哪个平台的 API Key？

**A**: 
- **新手用户**：推荐使用硅基流动（免费）
- **付费用户**：推荐使用 DeepSeek（性价比高）
- **高质量需求**：推荐使用 GPT-4o

## 使用相关

### Q8: 生成的周报不符合要求怎么办？

**A**: 可以尝试：
1. 调整 Prompt 模板
2. 修改 Daily Log 格式，使其更清晰
3. 手动编辑生成结果
4. 创建自定义模板

### Q9: 可以生成多周的周报吗？

**A**: 可以。选择更大的日期范围即可，但建议一次生成一周的内容，质量会更好。

### Q10: Daily Log 格式必须固定吗？

**A**: 不是必须的，但建议保持格式一致，有助于 AI 更好地理解内容，生成更准确的周报。

### Q11: 可以批量导入历史记录吗？

**A**: 目前不支持批量导入，但可以在首页直接粘贴多日内容生成周报。

### Q12: 记录的内容会丢失吗？

**A**: 不会。所有内容都会自动保存到云端，刷新页面也不会丢失。

## 功能相关

### Q13: 提醒功能不工作怎么办？

**A**: 请检查：
1. 配置信息是否正确（Webhook、密钥等）
2. 网络连接是否正常
3. 是否在工作日（节假日不会推送）
4. 点击「测试推送」验证配置

### Q14: 可以导出为其他格式吗？

**A**: 目前支持复制为文本，你可以粘贴到其他工具（如 Word、Notion、语雀）中转换为所需格式。

### Q15: 可以删除历史记录吗？

**A**: 可以。在「每日记录」页面选择日期后可以编辑或清空内容。

### Q16: 支持多语言吗？

**A**: 目前主要支持中文，未来可能会支持更多语言。

## 技术相关

### Q17: 生成失败怎么办？

**A**: 请检查：
1. API Key 是否配置正确
2. 网络连接是否正常
3. 模型服务是否可用
4. 查看错误提示信息

### Q18: 页面加载慢怎么办？

**A**: 可能的原因：
1. 网络连接较慢
2. 数据量较大
3. 服务器负载较高

建议：
- 检查网络连接
- 清除浏览器缓存
- 稍后重试

### Q19: 支持移动端吗？

**A**: 支持。WeekNote 采用响应式设计，可以在手机、平板等设备上使用。

1. 在 GitHub 上提交 Issue

### Q21: WeekNote 是免费的吗？

**A**: 不会。我们采用加密存储，数据安全有保障。你的 Daily Log 和生成的周报只有你可以看到。
