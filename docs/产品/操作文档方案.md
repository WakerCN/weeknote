## 📸 截图获取方案

### 方案一：使用浏览器自动化工具（推荐）✨

**优势**：
- ✅ 自动化批量截图，效率高
- ✅ 可以精确控制截图区域
- ✅ 支持不同分辨率截图
- ✅ 可以录制操作流程

**工具推荐**：
1. **Cursor Browser Extension**（已集成）
   - 使用 `browser_take_screenshot` 工具
   - 可以精确控制截图时机和区域
   - 支持全页面截图和元素截图

2. **Playwright**（命令行工具）
   ```bash
   # 安装
   npm install -g playwright
   
   # 截图示例
   playwright screenshot --full-page http://localhost:5173 ./screenshot.png
   ```

3. **Puppeteer**（Node.js 脚本）
   ```javascript
   const puppeteer = require('puppeteer');
   const browser = await puppeteer.launch();
   const page = await browser.newPage();
   await page.goto('http://localhost:5173');
   await page.screenshot({ path: 'screenshot.png', fullPage: true });
   ```

### 方案二：手动截图 + 标注工具

**工具推荐**：
1. **macOS 自带截图工具**
   - `⌘ + Shift + 4`：选择区域截图
   - `⌘ + Shift + 3`：全屏截图
   - `⌘ + Shift + 5`：截图工具（支持标注）

2. **标注工具**
   - **Snagit**：专业截图和标注工具
   - **CleanShot X**：macOS 截图工具，支持标注
   - **Annotate**：轻量级标注工具
   - **Markdown 图片标注**：使用 HTML `<img>` 标签 + CSS

### 方案三：使用 AI 生成截图说明

**工具推荐**：
- **Cursor Browser Extension**：可以自动生成操作说明
- **Figma**：设计稿截图（如果有设计稿）

## 🎥 视频录制方案

### 方案一：屏幕录制工具

**macOS 推荐工具**：
1. **QuickTime Player**（系统自带）
   - 文件 → 新建屏幕录制
   - 支持全屏或选择区域录制
   - 导出为 `.mov` 格式

2. **ScreenFlow**（付费）
   - 专业屏幕录制和编辑工具
   - 支持添加标注、箭头、高亮等

3. **Loom**（免费/付费）
   - 在线屏幕录制工具
   - 自动生成分享链接
   - 支持添加字幕

4. **OBS Studio**（免费）
   - 开源屏幕录制工具
   - 功能强大，适合专业录制

### 方案二：浏览器扩展录制

**推荐扩展**：
- **Loom Chrome Extension**：一键录制浏览器标签页
- **Screencastify**：Chrome 扩展，支持录制和编辑

### 视频处理建议

1. **视频格式**：推荐使用 `.mp4` 格式（兼容性好）
2. **视频长度**：单个操作视频建议 30-60 秒
3. **视频大小**：压缩后控制在 10MB 以内
4. **视频托管**：
   - **GitHub Releases**：上传到 GitHub Releases
   - **Bilibili/YouTube**：上传到视频平台，嵌入链接
   - **云存储**：阿里云 OSS、腾讯云 COS 等
